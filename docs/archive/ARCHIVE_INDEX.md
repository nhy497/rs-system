# 📚 RS-System 歷史文檔索引

> **最後更新**: 2026-02-12  
> **文檔命名規則**: `YYYYMMDD_HHmm_內容標題.md`

---

## 📌 命名規則說明

### 格式
```
YYYYMMDD_HHmm_標題.md
```

### 範例
- `20260121_0226_登入系統重構.md`
- `20260124_1825_測試系統改進.md`
- `20260125_0638_UTF8編碼修復.md`

### 時區
所有時間均使用 **UTC+8 (香港時間)**

---

## 📚 文檔分類

### 🐛 Bug 修復記錄 (bug-fixes/)
記錄各種 Bug 的診斷和修復過程

### 📋 測試報告 (reports/)
QA 測試、驗證報告、測試執行記錄

### 🚀 實現記錄 (implementations/)
功能實現、系統升級、技術集成的詳細記錄

### 📜 過時文檔 (legacy/)
不再使用但保留作參考的歷史文檔

---

## 🔍 快速查找

### 按日期查找
```bash
# 查看 2026-01-21 的所有文檔
ls docs/archive/**/20260121_*

# 查看 2026-01 整個月的文檔
ls docs/archive/**/202601*
```

### 按類型查找
```bash
# Bug 修復
ls docs/archive/bug-fixes/

# 測試報告
ls docs/archive/reports/

# 實現記錄
ls docs/archive/implementations/
```

### 按關鍵字搜尋
```bash
# 搜尋登入相關文檔
grep -r "登入" docs/archive/

# 搜尋 PouchDB 相關
grep -r "PouchDB" docs/archive/
```

## 📝 文檔維護

### 新增文檔
1. 按命名規則創建檔案
2. 放到對應的分類目錄
3. 更新本索引檔案

### 歸檔舊文檔
1. 重命名為新格式
2. 移動到 archive/ 對應目錄
3. 更新索引記錄

### 刪除文檔
- 不建議刪除
- 如需刪除，先檢查是否有引用
- 在索引中標記為「已刪除」
