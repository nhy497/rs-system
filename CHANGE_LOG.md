## 📝 使用說明

### 如何新增變更記錄

1. **複製模板**
   ```
   複製 CHANGE_TEMPLATE.md
   重命名為 CHANGE_[YYYYMMDD]_[功能名稱].md
   ```

2. **填入內容**
   ```
   按模板中的各部分填入實際內容
   包括：變更詳情、實現細節、測試報告
   ```

3. **更新此索引**
   ```
   在「已完成」或「進行中」或「待開發」中添加記錄
   按日期排序
   ```

4. **保存存檔**
   ```
   永久保留，放在與其他文檔同級目錄
   不刪除、不存檔、便於查詢
   ```

## 🎓 最佳實踐

### 變更文檔應包含
✅ 清晰的變更概述（問題 → 解決方案）

❌ 遺漏測試項目（全部記錄）  
❌ 不更新索引（及時同步）
