# 📚 使用者指南

## 系統簡介

RS-System 是一個輕量級的跳繩課堂記錄系統，專為教練設計，提供快速輸入、本地儲存、CSV 匯出等功能。

## 主要功能

### ✅ 快速輸入
直覺的表單設計，快速記錄課堂資訊

### ✅ 本地儲存
使用 localStorage 和 IndexedDB 雙重備份，確保資料安全

### ✅ CSV 匯出
一鍵匯出所有記錄，方便資料分析

### ✅ 班別管理
按班別分組查看和管理記錄

### ✅ 歷史記錄
快速重溫歷史課程

### ✅ 響應式設計
支持電腦和移動裝置

### ✅ 離線使用
無需網路連線即可使用

## 快速開始

### 1. 訪問系統

線上版本：https://nhy497.github.io/rs-system/

或本地開發：
```bash
npm run dev
```
訪問 `http://localhost:5173`

### 2. 登入系統

#### 預設 Creator 帳號
- 用戶名: `creator`
- 密碼: `1234`
- 權限: 完整系統權限

#### 註冊新帳號
1. 點擊登入頁面的「註冊」連結
2. 填寫用戶名、郵箱、密碼
3. 點擊「註冊」
4. 註冊成功後自動登入

### 3. 建立課堂記錄

1. 點擊「新增記錄」按鈕
2. 填寫課堂資訊：
   - **日期**: 選擇上課日期
   - **班級**: 輸入班級名稱
   - **學生人數**: 輸入人數
   - **技能**: 勾選學生完成的技能
   - **備註**: 添加課堂備註（選填）
3. 點擊「保存」

### 4. 查看記錄

#### 所有記錄
點擊側邊欄「記錄列表」查看所有課堂記錄

#### 按班別查看
使用篩選功能選擇特定班級

#### 搜尋記錄
使用搜尋框輸入關鍵字搜尋

### 5. 編輯記錄

1. 在記錄列表中找到要編輯的記錄
2. 點擊「編輯」按鈕
3. 修改資訊
4. 點擊「更新」保存變更

### 6. 刪除記錄

1. 在記錄列表中找到要刪除的記錄
2. 點擊「刪除」按鈕
3. 確認刪除

### 7. 匯出資料

1. 點擊「匯出」按鈕
2. 系統自動下載 CSV 檔案
3. 使用 Excel 或其他工具開啟

## 用戶角色

### Creator（創作者）

**權限**:
- ✅ 查看所有用戶資料
- ✅ 查看所有課程記錄
- ✅ 管理用戶帳戶
- ✅ 刪除用戶
- ✅ 查看系統統計
- ❌ 不能新增課程記錄（僅限查看）

**使用說明**:

#### 查看用戶課程
1. 登入 Creator 帳號
2. 點擊側邊欄「用戶管理」
3. 在用戶列表中點擊「📋 查看課程」
4. 查看該用戶的所有課程記錄
5. 點擊「查看詳情」查看完整資訊

#### 刪除用戶
1. 在「用戶管理」頁面找到目標用戶
2. 點擊「刪除」按鈕
3. 確認刪除（無法刪除當前登入的帳戶）

#### 查看系統統計
在「用戶管理」頁面查看：
- 總用戶數
- 總記錄數
- 系統使用統計

### User（一般用戶）

**權限**:
- ✅ 新增課程記錄
- ✅ 編輯自己的記錄
- ✅ 刪除自己的記錄
- ✅ 查看自己的記錄
- ✅ 匯出自己的資料
- ❌ 無法查看其他用戶的資料
- ❌ 無法訪問用戶管理頁面

## 常見問題 (FAQ)

### Q1: 如何備份我的資料？

**A**: 系統使用雙重備份機制：
1. localStorage（即時儲存）
2. IndexedDB（持久化儲存）

建議定期使用「匯出」功能下載 CSV 備份。

### Q2: 資料會丟失嗎？

**A**: 資料儲存在瀏覽器本地，除非：
- 清除瀏覽器資料
- 卸載瀏覽器
- 使用無痕模式

建議定期匯出 CSV 備份。

### Q3: 可以在多個裝置使用嗎？

**A**: 目前每個裝置的資料是獨立的。如需同步功能，請關注未來版本更新。

### Q4: 忘記密碼怎麼辦？

**A**: 目前系統沒有密碼重置功能。如果忘記密碼：
1. 使用瀏覽器開發工具清除 localStorage
2. 重新註冊帳號
3. 注意：清除資料會導致記錄遺失，請先匯出備份

### Q5: 如何聯絡支援？

**A**: 
1. 查看 [GitHub Issues](https://github.com/nhy497/rs-system/issues)
2. 創建新 Issue 描述問題
3. 提供錯誤截圖和步驟

### Q6: 系統支援哪些瀏覽器？

**A**: 支援現代瀏覽器：
- ✅ Chrome (推薦)
- ✅ Firefox
- ✅ Edge

- ❌ IE 11（不支援）

### Q7: 如何提供功能建議？

2. 點擊「New Issue」

4. 描述您的建議

## 疑難排解

### 問題：無法登入

**可能原因**:
- 用戶名或密碼錯誤
- localStorage 已滿
- 瀏覽器禁用 localStorage

**解決方法**:
1. 檢查用戶名和密碼
2. 清除瀏覽器快取
3. 檢查瀏覽器設定

### 問題：記錄無法保存

**可能原因**:
- localStorage 已滿（通常 5-10MB 限制）
- 瀏覽器禁用儲存

**解決方法**:
1. 匯出舊記錄並清理
2. 檢查瀏覽器儲存設定
3. 使用其他瀏覽器

### 問題：匯出失敗

**可能原因**:
- 瀏覽器阻擋下載
- 沒有記錄可匯出

**解決方法**:
1. 檢查瀏覽器下載設定
2. 確認有記錄存在
3. 嘗試其他瀏覽器

### 問題：頁面無法載入

**可能原因**:
- 網路問題
- JavaScript 被禁用
- 快取問題

**解決方法**:
1. 檢查網路連線
2. 啟用 JavaScript
3. 清除瀏覽器快取
4. 硬刷新（Ctrl+Shift+R）
